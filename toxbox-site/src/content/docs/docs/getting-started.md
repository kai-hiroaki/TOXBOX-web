このページでは、TOXBOX を入手して最初のビジュアル出力までの手順を説明します。

## 動作環境

- **TouchDesigner**: 2025.32050 以降
- **OS**: Windows / macOS（TouchDesigner が動作する環境）
- **GPU**: TouchDesigner の推奨スペックに準拠

:::note
TOXBOX は TouchDesigner 2025.32050 で制作されています。それ以前のバージョンでは正常に動作しない可能性があります。
:::

## ダウンロード

1. [BOOTH](https://simijimishijimi.booth.pm/items/7592348) から TOXBOX を購入・ダウンロード
2. ダウンロードした ZIP ファイルを任意の場所に解凍

解凍したフォルダ内の `TOXBOX.toe` をダブルクリックして TouchDesigner で開きます。

## 最初のビジュアル出力

### 1. tox ファイルを読み込む

TOXBOX に同梱されているサンプル tox をシーンパネルにドラッグ＆ドロップします。

サンプル tox は TOXBOX/tox にあります。

<!-- TODO: スクリーンショット - ドラッグ＆ドロップ -->

### 2. シーンを選択する

読み込んだシーンのセルをクリックして選択状態にします。選択されたシーンがプレビュー／出力されます。

### 3. レイヤーフェーダーで調整

レイヤーフェーダーを上げて、ビジュアルの透明度（ミックス量）を調整します。
