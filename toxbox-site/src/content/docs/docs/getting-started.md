## このページのゴール

- TOXBOX を入手して起動する
- 最初のビジュアル出力まで到達する

## 手順（TODO）

1. Booth から入手（ダウンロード）
2. TouchDesigner でプロジェクトを開く（制作: ver.2025.32050）
3. tox ファイルをドラッグ＆ドロップで読み込む
4. レイヤーフェーダーで透明度を調整してミックスする

## 最初に試すこと

- **tox 読み込み**: 手持ちのシーン（tox）を読み込んで出力を確認
- **Audio / Sequencer**: 解析・シーケンス信号をパラメータに参照して動作を確認
- **MIDI**: 手元の MIDI コントローラーで簡単に割り当てて操作性を確認
