## 主な機能

- tox ファイルの読み込み（ドラッグ＆ドロップ）
- レイヤー（4 レイヤー / 各 8 = 合計 32）
  - フェーダーで透明度調整
  - 最上段レイヤーはマスク指定可能
- オーディオ解析（Audio Device In CHOP）
  - Low_Trigger / Low / High_Trigger / High / Peak
- ステップシーケンサー（BPM 連動）
  - Boolean / Attack / Filter
- パラメータパネル
  - 指定形式の tox を作ることで、選択 tox のパラメータをパフォーマンスモードで調整
  - AudioAnalysis / Sequencer / Color などの値を ChopReference
- ポストエフェクト（8 種類）
- MIDI マッピング
