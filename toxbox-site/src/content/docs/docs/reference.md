このページでは、TOXBOX の全機能とパラメータを体系的にまとめています。

## 動作環境

| 項目 | 要件 |
|------|------|
| TouchDesigner | 2025.32050 以降 |
| OS | Windows 10/11、macOS（TouchDesigner 対応版） |
| GPU | TouchDesigner の推奨スペックに準拠 |
| RAM | 16GB 以上推奨 |

:::note
TOXBOX は TouchDesigner 2025.32050 で制作されています。以前のバージョンでは互換性の問題が発生する可能性があります。
:::

## Scene Panel

### 概要

シーン（tox）の管理と選択を行うパネル。

### 構成

| 項目 | 値 |
|------|-----|
| レイヤー数 | 4 |
| シーン数 / レイヤー | 8 |
| 合計シーン数 | 32 |

### 機能一覧

| 機能 | 説明 |
|------|------|
| tox 読み込み | ドラッグ＆ドロップで tox ファイルを読み込み |
| シーン選択 | クリックでシーンをアクティブ化 |
| レイヤーフェーダー | 0〜100% で透明度を調整 |
| マスク機能 | レイヤー 1 をマスクとして使用可能 |

### レイヤー重なり順

```
出力（最前面）
   ↑
レイヤー 1（マスク可能）
レイヤー 2
レイヤー 3
レイヤー 4（最背面）
```

## Parameter Panel

### 概要

選択中シーンのカスタムパラメータを調整するパネル。

### 対応パラメータタイプ

| タイプ | 表示 | 値の範囲 |
|--------|------|---------|
| Float | スライダー | カスタム定義 |
| Int | スライダー（整数） | カスタム定義 |
| Toggle | チェックボックス | 0 or 1 |
| Menu | ドロップダウン | 定義された選択肢 |
| RGB | カラーピッカー | 0〜1 (RGB) |
| RGBA | カラーピッカー | 0〜1 (RGBA) |

### ChopReference

パラメータに接続可能な信号源：

| ソース | 信号 |
|--------|------|
| Audio Analysis | Low_Trigger, Low, High_Trigger, High, Peak |
| Sequencer | Boolean, Attack, Filter |
| Color | R, G, B |

## Audio Analysis

### 概要

音声入力を解析し、ビジュアルを音楽に反応させる機能。

### 入力設定

| パラメータ | 説明 | デフォルト |
|------------|------|----------|
| Audio Device In | 入力デバイス選択 | システムデフォルト |
| Gain | 入力ゲイン | 1.0 |

### 出力信号

| 信号名 | タイプ | 値の範囲 | 説明 |
|--------|--------|---------|------|
| `Low_Trigger` | トリガー | 0 or 1 | 低音域のトリガー |
| `Low` | レベル | 0〜1 | 低音域のレベル |
| `High_Trigger` | トリガー | 0 or 1 | 高音域のトリガー |
| `High` | レベル | 0〜1 | 高音域のレベル |
| `Peak` | レベル | 0〜1 | 全体のピークレベル |

### 調整パラメータ

| パラメータ | 説明 | 範囲 |
|------------|------|------|
| Threshold (Low) | 低音トリガーの閾値 | 0〜1 |
| Threshold (High) | 高音トリガーの閾値 | 0〜1 |
| Smoothing | レベル信号のスムージング | 0〜1 |

## Sequencer

### 概要

BPM に同期したステップシーケンサー。

### 基本パラメータ

| パラメータ | 説明 | 範囲 |
|------------|------|------|
| BPM | テンポ | 20〜300 |
| Steps | ステップ数 | 4, 8, 16, 32 |

### 出力モード

| モード | 出力特性 | 用途 |
|--------|----------|------|
| Boolean | 0 or 1 | オン/オフ切り替え |
| Attack | 急上昇→減衰 | パルス的な変化 |
| Filter | スムーズな補間 | 緩やかな変化 |

### Attack モード パラメータ

| パラメータ | 説明 | 範囲 |
|------------|------|------|
| Attack | 立ち上がり時間 | 0〜1 |
| Decay | 減衰時間 | 0〜1 |

### Filter モード パラメータ

| パラメータ | 説明 | 範囲 |
|------------|------|------|
| Smoothing | 補間のスムーズさ | 0〜1 |

## Post Effects

### 概要

最終出力に適用されるエフェクト。8 種類を搭載。

### Blur（ブラー）

| パラメータ | 説明 | 範囲 |
|------------|------|------|
| Amount | ぼかしの強さ | 0〜1 |

### Bloom（ブルーム）

| パラメータ | 説明 | 範囲 |
|------------|------|------|
| Intensity | 発光の強さ | 0〜1 |
| Threshold | 発光閾値 | 0〜1 |

### Chromatic Aberration（色収差）

| パラメータ | 説明 | 範囲 |
|------------|------|------|
| Amount | 色ズレの量 | 0〜1 |

### Feedback（フィードバック）

| パラメータ | 説明 | 範囲 |
|------------|------|------|
| Amount | フィードバック量 | 0〜1 |
| Decay | 減衰速度 | 0〜1 |

### Edge（エッジ）

| パラメータ | 説明 | 範囲 |
|------------|------|------|
| Strength | エッジの強さ | 0〜1 |

### Mosaic（モザイク）

| パラメータ | 説明 | 範囲 |
|------------|------|------|
| Size | モザイクサイズ | 1〜100 |

### RGB Shift（RGB シフト）

| パラメータ | 説明 | 範囲 |
|------------|------|------|
| Amount | シフト量 | 0〜1 |
| Angle | シフト方向 | 0〜360 |

### Invert（反転）

| パラメータ | 説明 | 範囲 |
|------------|------|------|
| Amount | 反転度合い | 0〜1 |

## MIDI Mapping

### 概要

MIDI コントローラーを TOXBOX の各機能に割り当てる機能。

### 対応 MIDI メッセージ

| メッセージタイプ | 用途 |
|-----------------|------|
| Control Change (CC) | ノブ、フェーダー |
| Note On/Off | ボタン、パッド |

### マッピング可能な対象

| カテゴリ | 対象 |
|----------|------|
| Scene Panel | シーン選択、レイヤーフェーダー |
| Parameter Panel | カスタムパラメータ |
| Sequencer | BPM、Tap Tempo |
| Audio Analysis | Gain、Threshold |
| Post Effects | エフェクト強度、On/Off |

## 出力

### 出力解像度

| プリセット | 解像度 |
|------------|--------|
| 720p | 1280 × 720 |
| 1080p | 1920 × 1080 |
| 4K | 3840 × 2160 |
| カスタム | 任意 |

### 出力先

| 出力方法 | 説明 |
|----------|------|
| ウィンドウ出力 | セカンドモニター等に表示 |
| NDI | ネットワーク経由で映像送信 |
| Spout/Syphon | 他アプリケーションへの映像共有 |

## キーボードショートカット

| キー | 機能 |
|------|------|
| F1 | パフォーマンスモード切り替え |
| 1-8 | レイヤー 1 のシーン 1-8 を選択 |
| Space | シーケンサー再生/停止 |

## ファイル形式

### 対応ファイル

| 形式 | 用途 |
|------|------|
| .tox | TouchDesigner Component File（シーン） |
| .toe | TouchDesigner Project File（プロジェクト） |

### プロジェクト構成

```
TOXBOX/
├── TOXBOX.toe          # メインプロジェクト
├── toxes/              # サンプル tox
│   ├── sample1.tox
│   ├── sample2.tox
│   └── ...
└── README.txt          # 説明書
```
