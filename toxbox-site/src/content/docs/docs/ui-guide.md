TOXBOX のパフォーマンスモード画面は、複数のパネルで構成されています。このページでは各パネルの役割と基本的な操作方法を説明します。

## 画面構成の概要

TOXBOX は以下のパネルで構成されています：

1. **Scene Panel**: シーン（tox）の管理と選択
2. **Parameter Panel**: 選択中シーンのパラメータ調整
3. **Audio Analysis Panel**: オーディオ入力の解析と信号出力
4. **Sequencer Panel**: BPM 同期のシーケンサー
5. **Post Effects Panel**: 最終出力へのエフェクト適用
6. **MIDI Mapping**: MIDI コントローラーの割り当て

## Scene Panel

シーンの読み込みと選択を行う、TOXBOX の中心となるパネルです。

### 構成

- **4 レイヤー**: 縦に 4 つのレイヤーが並ぶ
- **8 シーン / レイヤー**: 各レイヤーに 8 つのシーンスロット
- **合計 32 シーン**: 4 × 8 = 32 シーンを同時に管理可能

### 基本操作

| 操作 | 説明 |
|------|------|
| ドラッグ＆ドロップ | tox ファイルをシーンスロットに読み込む |
| クリック | シーンを選択（アクティブ化） |
| フェーダー | レイヤーの透明度（ミックス量）を調整 |

### レイヤーミキシング

4 つのレイヤーは上から下に重なって表示されます：

- **レイヤー 1（最上段）**: 最前面に表示、マスク機能あり
- **レイヤー 2**: レイヤー 1 の下に表示
- **レイヤー 3**: レイヤー 2 の下に表示
- **レイヤー 4（最下段）**: 最背面に表示

各レイヤーのフェーダーで透明度を調整し、複数のビジュアルをミックスできます。

### マスク機能

最上段のレイヤー（レイヤー 1）はマスクとして使用できます。白黒のビジュアルを読み込むと、白い部分は透過、黒い部分は遮蔽として機能します。

## Parameter Panel

選択中のシーンのパラメータを調整するパネルです。

### 対応シーン

パラメータパネルに対応した形式で作成された tox のみ、パラメータが表示されます。対応形式については [tox作成ガイド](../creating-tox/) を参照してください。

### 操作方法

- パラメータ名の横のスライダーやノブで値を調整
- 数値入力欄に直接値を入力することも可能

### ChopReference

パラメータには以下の信号を参照（ChopReference）として接続できます：

- **Audio Analysis**: Low, High, Peak などの音声解析値
- **Sequencer**: シーケンサーの出力値
- **Color**: カラーピッカーからの RGB 値

## Audio Analysis Panel

オーディオ入力を解析し、ビジュアルを音楽に反応させるためのパネルです。

### 入力設定

**Audio Device In** で入力デバイスを選択します。システムのオーディオ出力をループバックする設定も可能です。

### 出力信号

| 信号名 | 説明 |
|--------|------|
| `Low_Trigger` | 低音域のトリガー（0 or 1） |
| `Low` | 低音域のレベル（0〜1） |
| `High_Trigger` | 高音域のトリガー（0 or 1） |
| `High` | 高音域のレベル（0〜1） |
| `Peak` | 全体のピークレベル（0〜1） |

### パラメータへの接続

これらの信号を tox のパラメータに接続することで、音楽に反応するビジュアルを作成できます。詳細は [オーディオ解析チュートリアル](../tutorials/audio-analysis/) を参照してください。

## Sequencer Panel

BPM に同期したシーケンサーで、リズミカルな変化を自動生成します。

### BPM 設定

シーケンサーの動作速度を BPM（Beats Per Minute）で設定します。手動入力または Tap Tempo で設定可能です。

### シーケンサーモード

3 つのモードがあり、それぞれ異なる出力特性を持ちます：

| モード | 説明 |
|--------|------|
| **Boolean** | 0 または 1 のオン/オフ信号 |
| **Attack** | トリガー時に急上昇し、徐々に減衰 |
| **Filter** | スムーズに補間された連続値 |

### パラメータへの接続

シーケンサーの出力を tox のパラメータに接続することで、BPM に同期した自動変化を作成できます。詳細は [シーケンサーチュートリアル](../tutorials/sequencer/) を参照してください。

## Post Effects Panel

最終出力にエフェクトを適用するパネルです。

### エフェクト一覧

TOXBOX には 8 種類のポストエフェクトが搭載されています：

1. **Blur**: ぼかし効果
2. **Bloom**: 明るい部分の発光
3. **Chromatic Aberration**: 色収差
4. **Feedback**: フィードバック効果
5. **Edge**: エッジ検出
6. **Mosaic**: モザイク
7. **RGB Shift**: RGB ずらし
8. **Invert**: 色反転

### 操作方法

各エフェクトのオン/オフと強度を調整できます。複数のエフェクトを同時に適用することも可能です。

詳細は [ポストエフェクトチュートリアル](../tutorials/post-effects/) を参照してください。

## MIDI Mapping

MIDI コントローラーを TOXBOX の各機能に割り当てます。

### 対応デバイス

一般的な MIDI コントローラー（ノブ、フェーダー、ボタン）に対応しています。

### マッピング手順

1. MIDI コントローラーを接続
2. MIDI Mapping モードを有効化
3. マッピングしたい TOXBOX のコントロールをクリック
4. MIDI コントローラーの対応するノブ/フェーダー/ボタンを操作
5. マッピングが完了

詳細は [MIDIマッピングチュートリアル](../tutorials/midi-mapping/) を参照してください。
