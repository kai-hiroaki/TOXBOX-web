このチュートリアルでは、TOXBOX のポストエフェクト機能を使って、最終出力にエフェクトを適用する方法を学びます。

## ポストエフェクトとは

ポストエフェクトは、すべてのレイヤーがミックスされた後の**最終出力**に適用されるエフェクトです。シーン（tox）の内容に関係なく、出力全体にエフェクトをかけることができます。

## エフェクト一覧

TOXBOX には 8 種類のポストエフェクトが搭載されています。

### 1. Blur（ブラー）

画面全体をぼかすエフェクトです。

| パラメータ | 説明 |
|------------|------|
| Amount | ぼかしの強さ |

**使いどころ**: 柔らかい雰囲気、フォーカス効果、トランジション

### 2. Bloom（ブルーム）

明るい部分が発光して周囲ににじむエフェクトです。

| パラメータ | 説明 |
|------------|------|
| Intensity | 発光の強さ |
| Threshold | 発光する明るさの閾値 |

**使いどころ**: 光の強調、幻想的な雰囲気、ネオン感

### 3. Chromatic Aberration（色収差）

レンズの歪みによる RGB のズレを再現するエフェクトです。

| パラメータ | 説明 |
|------------|------|
| Amount | ズレの量 |

**使いどころ**: アナログ感、VHS 風、グリッチ

### 4. Feedback（フィードバック）

前フレームの映像を合成し続けるエフェクトです。残像効果が得られます。

| パラメータ | 説明 |
|------------|------|
| Amount | フィードバックの量 |
| Decay | 減衰の速さ |

**使いどころ**: 残像、トレイル効果、サイケデリック

:::caution
Feedback を強くかけすぎると、映像が飽和して真っ白/真っ黒になることがあります。
:::

### 5. Edge（エッジ）

輪郭線を検出して強調するエフェクトです。

| パラメータ | 説明 |
|------------|------|
| Strength | エッジの強さ |

**使いどころ**: アウトライン、スケッチ風、構造の強調

### 6. Mosaic（モザイク）

画面をピクセル状のブロックに分割するエフェクトです。

| パラメータ | 説明 |
|------------|------|
| Size | モザイクのサイズ |

**使いどころ**: ピクセルアート風、解像度の演出、テクスチャ

### 7. RGB Shift（RGB シフト）

RGB の各チャンネルを別々にずらすエフェクトです。

| パラメータ | 説明 |
|------------|------|
| Amount | シフト量 |
| Angle | シフトの方向 |

**使いどころ**: グリッチ、3D アナグリフ風、VHS エラー

### 8. Invert（反転）

色を反転させるエフェクトです。

| パラメータ | 説明 |
|------------|------|
| Amount | 反転の度合い（0: 通常、1: 完全反転） |

**使いどころ**: ネガポジ反転、フラッシュ効果、インパクト

## 基本操作

### エフェクトのオン/オフ

各エフェクトの横にあるトグルボタンでオン/オフを切り替えます。

### 強度の調整

スライダーまたはノブでエフェクトの強度を調整します。

### 複数エフェクトの適用

複数のエフェクトを同時にオンにすることで、エフェクトを重ねがけできます。

:::note
エフェクトは上から順に適用されます。順序によって結果が変わることがあります。
:::

## 実践例

### 例 1: ドロップでのインパクト

曲のドロップに合わせて強いインパクトを出す設定：

1. **Bloom**: Intensity を高めに設定
2. **Chromatic Aberration**: 軽くかける
3. ドロップ時に両方をオン → 強い光と歪みで盛り上がりを演出

### 例 2: アンビエント/チルな雰囲気

穏やかな曲調に合う柔らかい演出：

1. **Blur**: 軽くぼかす
2. **Feedback**: Amount を低め、Decay を遅めに設定
3. → 夢のようなふんわりした雰囲気

### 例 3: グリッチ/サイバー

デジタルノイズ感のある演出：

1. **Chromatic Aberration**: 強めにかける
2. **RGB Shift**: 小さめのシフト量
3. **Mosaic**: 細かめのサイズ
4. → デジタルエラーのような視覚効果

### 例 4: レトロ/VHS 風

古いビデオテープのような演出：

1. **Chromatic Aberration**: 軽く
2. **Feedback**: 少量の残像
3. **RGB Shift**: 微妙にずらす
4. → アナログビデオ風の質感

## オーディオ/シーケンサーとの連携

ポストエフェクトのパラメータも ChopReference で接続できます。

### オーディオ解析との連携

- **Peak → Bloom Intensity**: 音量に応じて発光
- **Low_Trigger → Invert Amount**: キックに反応して色反転

### シーケンサーとの連携

- **Attack → Chromatic Aberration**: リズムに合わせてグリッチ
- **Filter → Blur Amount**: ゆっくりとぼかしが変化

## MIDI での操作

ポストエフェクトは MIDI コントローラーでの操作に適しています。

推奨マッピング：
- **ノブ**: エフェクト強度の調整
- **ボタン**: エフェクトのオン/オフ

詳細は [MIDIマッピング](midi-mapping/) を参照してください。

## パフォーマンスへの影響

ポストエフェクトは GPU 処理を使用するため、多くのエフェクトを同時に適用すると負荷が増加します。

| 負荷 | エフェクト |
|------|-----------|
| 軽い | Invert, Edge, Mosaic |
| 中程度 | Chromatic Aberration, RGB Shift |
| 重い | Blur, Bloom, Feedback |

:::tip
パフォーマンスが低下する場合は、使用するエフェクト数を減らすか、Blur や Bloom の品質設定を下げてみてください。
:::
