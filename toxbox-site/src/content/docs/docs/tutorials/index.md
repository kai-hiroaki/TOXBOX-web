TOXBOX の各機能を実践的に学ぶためのチュートリアルです。

## チュートリアル一覧

### 基本操作

- **[tox読み込みとレイヤーミックス](loading-tox/)**: tox ファイルの読み込みから、4 レイヤーを使ったミキシングまでの基本操作

### 音楽連動

- **[オーディオ解析の使い方](audio-analysis/)**: 音楽に反応するビジュアルを作成するためのオーディオ解析機能
- **[シーケンサーの使い方](sequencer/)**: BPM に同期した自動変化を作成するシーケンサー機能

### コントローラー

- **[MIDIマッピング](midi-mapping/)**: MIDI コントローラーを使った操作の設定方法

### エフェクト

- **[ポストエフェクト](post-effects/)**: 最終出力に適用できる 8 種類のエフェクト

## 推奨する学習順序

1. まず **[tox読み込みとレイヤーミックス](loading-tox/)** で基本操作を習得
2. 次に **[オーディオ解析](audio-analysis/)** または **[シーケンサー](sequencer/)** で動的な演出を学ぶ
3. 現場で使う場合は **[MIDIマッピング](midi-mapping/)** でコントローラー設定
4. **[ポストエフェクト](post-effects/)** で仕上げの演出を追加
