## よくある問題（TODO）

- **起動しない / 開けない**
  - TouchDesigner のバージョン差による可能性があります（制作: ver.2025.32050）
- **画が出ない**
  - tox が読み込めているか（ドラッグ＆ドロップ後に選択状態になるか）を確認
  - レイヤーフェーダーが 0 になっていないか確認
- **重い / カクつく**
  - 未選択 tox の cook を切る仕組みで改善を狙っています。シーン数を減らして切り分けしてください
