# 非公開ドキュメント（開発 / Cursor 用）

この `docs/` 配下は **GitHub Pages で公開しない**、開発・運用・Cursor 作業用のドキュメント置き場です。  
公開用Docs（Starlight）は `toxbox-site/src/content/docs/` 配下に置き、ここには混ぜません。

## どれを読めば良いか

- **Cursor で作業するとき**: `cursor.md`
- **リポジトリの構成を把握したい**: `repo-structure.md`
- **ローカルで動かしたい**: `dev.md`
- **GitHub Pages デプロイを理解したい**: `deploy.md`
- **/releases の方針・実装メモ**: `releases.md`
